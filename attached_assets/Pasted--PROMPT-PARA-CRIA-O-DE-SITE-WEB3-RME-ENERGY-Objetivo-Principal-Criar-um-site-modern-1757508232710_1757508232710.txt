## **PROMPT PARA CRIAÇÃO DE SITE WEB3 - RME ENERGY**

**Objetivo Principal:** Criar um site moderno, de página única (single-page landing page), para o projeto de energia limpa "RME Energy", tokenizado na blockchain Solana. O site deve ser informativo, visualmente impactante e funcional, com integração para compra do token RME através da carteira Phantom.

**Tecnologias Sugeridas:**
* **Frontend:** React.js ou Next.js para uma estrutura moderna e dinâmica.
* **Estilização:** Tailwind CSS para um design futurista e responsivo.
* **Web3/Solana:** Bibliotecas `@solana/wallet-adapter` para conexão de carteiras e `@solana/web3.js` para interações com a blockchain.

**Requisitos Funcionais (Web3):**
1.  O botão **"Conectar Carteira"** deve usar o `@solana/wallet-adapter` para abrir um modal de seleção de carteira, com destaque para a Phantom.
2.  Após a conexão, o botão deve exibir o endereço da carteira do usuário (formato truncado, ex: `ABCD...wxyz`).
3.  Os botões **"Compre RME Agora"** e **"Compre RME na Raydium"** devem direcionar o usuário para a página de swap da Raydium (ou Jupiter) com o token RME já selecionado como moeda de saída. O link deve ser: `https://raydium.io/swap/?inputMint=sol&outputMint=HrSUCXgwQNVr4fhKbZxokLEdPskVJ2swwGXZkM4Hpump`

**Estilo e Design:**
* **Tema:** Dark (escuro), com detalhes em verde-limão vibrante, seguindo a identidade visual das imagens.
* **Tipografia:** Moderna e limpa (Ex: Poppins, Inter, ou similar).
* **Animações:** Sutis ao rolar a página para revelar as seções. Efeito "glow" (brilho) nos elementos verdes.
* **Responsividade:** O site deve ser totalmente funcional e visualmente agradável em desktops, tablets e celulares.
